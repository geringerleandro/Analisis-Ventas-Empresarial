## Escenario de Ventas
Proyecto enfocado en el análisis de datos de ventas diarias para generar indicadores que permitan interpretar el desempeño comercial de la empresa a lo largo del año 2024.

## Dataset utilizado
Archivo: sales_sample_2024.csv
Fuente: Dataset provisto por la cátedra de Organización Empresarial UTN.
Contenido: 365 registros de ventas diarias correspondientes al año 2024.
Columnas: id, sales_date, sales_amount.

## Indicadores que calcula el script
- Ventas totales del año
- Promedio diario de ventas
- Día con la venta más alta
- Mes con mayor volumen de ventas
- Gráfico de evolución de ventas diarias

## Estructura del Proyecto
- /data: Dataset original (CSV).
- /scripts: Script de procesamiento y análisis.
- /resultados: Gráficos y reportes finales.

## Instrucciones de uso
1. Clonar el repositorio.
2. Subir el archivo sales_sample_2024.csv a la carpeta /data en Google Colab.
3. Ejecutar el script analisis_ventas.py desde la carpeta /scripts usando Google Colab.
